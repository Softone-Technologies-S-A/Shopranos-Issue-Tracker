# Feeder

{% hint style="success" %}
Αναζητήστε Feeder (Σε Προσχέδιο, Αρχειοθετημένο, Ενεργό) μεταβαίνοντας: **Ρυθμίσεις > Εισαγωγή – Εξαγωγή Δεδομένων > Feeders.**
{% endhint %}

Στη _λίστα_ θα βρείτε: _Feeder, Κατάσταση, Ημερομηνία Εισαγωγής_ και _Ενέργειες._

<figure><img src="../../.gitbook/assets/ScreenHunter 95.png" alt=""><figcaption></figcaption></figure>

Επεξεργαστείτε Feeder αλλάζοντας αριστερά την _Κατάσταση_ (Ενεργό, Σε Προσχέδιο, Αρχειοθετημένο) και δεξιά το _Όνομα_, τη _Συνθήκη_ (Συνδυασμός συνθηκών, Χαρακτηρισμός Πηγής, Χαρακτηρισμός, Category Path, Κατηγορία, Χαρακτηριστικό, Κατασκευαστής, Τιμή) και τη _Λήξη_ (επιλέγοντας κάθε μία ώρα ή παραπάνω). Μπορείτε να αντιγράψετε τα Feeders ιστοτόπων, πατώντας δεξιά από το καθένα, το εικονίδιο αντιγραφής.

<figure><img src="../../.gitbook/assets/ScreenHunter 93.png" alt=""><figcaption></figcaption></figure>

Προσθέστε Feeder, ορίζοντας αριστερά την _Κατάσταση_ (Ενεργό, Σε Προσχέδιο, Αρχειοθετημένο) και δεξιά το _Όνομα_, τη _Συνθήκη_ (Συνδυασμός συνθηκών, Χαρακτηρισμός Πηγής, Χαρακτηρισμός, Category Path, Κατηγορία, Χαρακτηριστικό, Κατασκευαστής, Τιμή) και τη _Λήξη_ (επιλέγοντας κάθε μία ώρα ή παραπάνω). Ακριβώς από κάτω, εμφανίζονται τα Feeders ιστοτόπων που μπορείτε να αντιγράψετε, πατώντας δεξιά το εικονίδιο αντιγραφής.
