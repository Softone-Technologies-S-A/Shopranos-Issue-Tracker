# Αρχική Σελίδα

{% hint style="success" %}
Μεταβείτε: **Layout Design > Αρχική Σελίδα**.
{% endhint %}

Στην καρτέλα **<Πρότυπο>**, μπορείτε να προσθέσετε, επιλέγοντας: +Προσθήκη > Βασικά Components (π.χ. Κείμενο με εικόνα, Products Block, Εγγραφή σε Newsletter), ή να επεξεργαστείτε, πατώντας το εικονίδιο μολυβιού, components, ορίζοντας τη μορφή της Αρχικής Σελίδας του e-shop σας.&#x20;

<figure><img src="../.gitbook/assets/ScreenHunter 105.png" alt=""><figcaption></figcaption></figure>

Μπορείτε να διαγράψετε οριστικά τη σελίδα, επιλέγοντας το κουτάκι <Επιβεβαιώστε τη διαγραφή> και στη συνέχεια πατώντας <mark style="color:red;"><Διαγραφή σελίδας></mark>.
