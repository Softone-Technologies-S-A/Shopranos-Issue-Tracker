# Κατηγορίες

{% hint style="info" %}
Για να δείτε τη Λίστα Ομάδων Πελατών σας, μεταβείτε: **Πελάτες > Κατηγορίες.**.
{% endhint %}

Αναζητώντας ανάμεσα στα: Σε Προσχέδιο, Αρχειοθετημένο και Ενεργό, θα δείτε τις κατηγορίες __ (Παλαιότερο σε νεότερο, Νεότερο σε παλαιότερο, Αλφαβητικά Α-Ω, Αλφαβητικά Ω-Α).

Στα αποτελέσματα αναζήτησης θα εμφανιστούν: οι _Ομάδες Πελατών_, η _Κατάσταση_, η _Ημερομηνία εισαγωγής_, ενώ μπορείτε να επιλέξετε την εμφάνιση ενεργειών ή τη διαγραφή της ομάδας πελατών (Ενέργειες > Εμφάνιση ή Ενέργειες > Διαγραφή).
